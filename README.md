Docker instalation for SandMarket project for Windows 10.

## Installation

### Open PowerShell as Administrator!

* mkdir SandMarket
* git clone --recursive git@github.com:madonzy/sand-market-env-win10.git SandMarket
* cd SandMarket
* .\m2devbox-init.bat
* .\m2devbox-sample-data-init.bat
